# Pomodoro Timer

![IMG_20251106_194643](https://github.com/user-attachments/assets/ac2901e0-bfe1-4879-9a52-be1ff244c014)


A simple Pomodoro timer with 25 minute FOCUS and 5 minute BREAK stages. Can be paused and reset. Made with ESP32-C3, tiny OLED screen and a powerd by small Li-ion battery, charged via TP4056.
